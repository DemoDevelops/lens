# Security Policy

## Reporting a vulnerability

Please do not open a public issue for security problems.

Report privately through GitHub's [Report a vulnerability](https://github.com/DemoDevelops/lens/security/advisories/new)
flow (the **Security** tab on the repository). Include steps to reproduce and the
affected version (`lens --version`).

You can expect an initial response within a few days. Fixes ship in a patch release
and are credited in the release notes unless you ask otherwise.

## Supported versions

Only the latest release receives security fixes.
